Procurando um desenvolvedor FullStack?

Johan Garcia Disciplina, qualidade e consistência. Vamos evoluir, crescer e fazer a diferença juntos?

Eu me descreveria como um profissional esforçado, que busca sempre adquirir mais conhecimentos, visando sempre ter um bom nível de proatividade para realizar soluções de problemas com mais rapidez. Sou uma pessoa detalhista, que se importa com os mínimos detalhes, além disso, programar para mim é uma paixão, e com isso, tudo que está relacionado a programação eu vejo como uma oportunidade de aperfeiçoar sempre mais minhas habilidades de desenvolvimento de sistemas. Não tenho nenhum problema com comunicação, logo, naturalmente consigo participar de conversas/reuniões com meus colegas de trabalho. Ademais, estou sempre disposto a aprender coisas novas, e assim, expandir minhas habilidades em programação, que para mim, é uma verdadeira paixão.

Este foi o primeiro projeto prático usando apenas HTML e CSS com os conteúdos estudados no módulo 01 do curso Profissão Programador. Durante a contrução do web site foi possível por em prática diversos conteúdos vistos anteriormente de forma teórica, conforme o programado. Pude perceber que ao raciocinar para resolver certos problemas o aprendizado se torna muito mais eficiente. Com isso, concluo que aprendi bastante com esse projeto.
